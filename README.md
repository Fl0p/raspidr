# RaspiDR

Go application for Raspberry Pi Zero 2W.

## Requirements

- Go 1.25+
- Make

## Quick Start

```bash
# Run locally
make run

# Build for Raspberry Pi
make build-pi

# Deploy to Raspberry Pi
make deploy PI_HOST=raspberrypi.local
```
