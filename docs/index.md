# RaspiDR Documentation

## Overview

RaspiDR is a Go application designed for Raspberry Pi Zero 2W.

## Getting Started

### Build

```bash
# Local build
make build

# Build for Raspberry Pi (ARM64)
make build-pi
```

### Deploy

```bash
make deploy PI_HOST=raspberrypi.local PI_USER=pi
```
